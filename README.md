# C* Programming Language

C* is a systems programming language that prioritizes
explicitness over brevity, determinism over convenience,
and compiler guarantees over runtime hope.

## Core Philosophy

If it compiled — it works.

Every variable has an explicit owner.
Every error has an explicit handler.
Every operation has an explicit meaning.
The compiler guarantees all of this before your program runs.

## Status

Currently in specification phase.
Compiler implementation starting summer 2026.
